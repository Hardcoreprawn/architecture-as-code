# Commands to push to GitHub

# After creating the repo on GitHub, run these commands:

# Add the remote (replace YOUR_USERNAME and REPO_NAME)
git remote add origin https://github.com/YOUR_USERNAME/azure-architecture-scanner.git

# Push to GitHub
git push -u origin master

# Done! Your repo is now live at:
# https://github.com/YOUR_USERNAME/azure-architecture-scanner
